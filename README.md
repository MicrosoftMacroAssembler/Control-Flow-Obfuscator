# Control-Flow-Obfuscator
Kernel Driver that includes varying Code Obfuscation Features for Anti-Reverse Engineering <br />
Supports all Compilers included in Visual Studio's 2026 <br />

# Obfuscator Features
Core Obfuscation Features:
- Indirect branching
- Segfault triggers
- Fake PE section signature (Denuvo, VMProtect, Themida, Enigma, SecuROM)
- Loop obfuscation
- Function Call Hiding
- String Protection
